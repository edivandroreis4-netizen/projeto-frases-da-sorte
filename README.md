# Projeto Frases da Sorte

Projeto desenvolvido em Portugol Studio para praticar lógica de programação usando estruturas condicionais.

## Funcionalidade

O programa solicita ao usuário o dia do seu aniversário e exibe uma frase da sorte correspondente ao dia informado.

## Tecnologias utilizadas

- Portugol Studio
- Lógica de programação
- Estrutura `escolha/caso`

## Como funciona

1. O usuário informa um número de 1 a 31.
2. O programa verifica o número informado.
3. Caso o número seja válido, exibe uma frase da sorte.
4. Caso o número seja inválido, exibe uma mensagem de erro.

## Como executar

1. Abra o arquivo `projeto-frases-da-sorte.por` no Portugol Studio.
2. Clique em executar.
3. Informe um dia entre 1 e 31.
4. Veja a frase da sorte exibida no console.

## Objetivo do projeto

Este projeto foi criado para praticar:

- Entrada de dados
- Condicionais
- Estrutura `escolha/caso`
- Organização de código
- Tratamento de entrada inválida
